# 6-DOF Robotic Arm — ROS2 & MoveIt
![IMG_4833](https://github.com/user-attachments/assets/425a2e69-06a6-4667-a3d7-5afb3c5dcbca)
A fully functional 6-axis robotic arm, 3D printed and built 
from scratch as a hands-on learning project.

## Inspiration

Based on the open-source Skyentific robotic arm project, 
adapted and rebuilt independently.

## What I changed

Not all original mechanical parts were available — motors 
and power transmission belts from the original design 
couldn't be sourced locally. I redesigned several gears 
and transmission components to work with accessible 
alternatives.

## Electronics

Built the electronics layer by studying circuit board 
schematics and following various tutorials. No prior 
electronics background — learned through doing.

## Software & Simulation

- Followed ROS2 and MoveIt official documentation to 
  set up the robot simulation
- Built the URDF model using Onshape
- Integrated Gazebo for simulation and MoveIt for 
  motion planning


## Purpose

This project was intentionally a learning journey — 
not a commercial product. The goal was to go through 
the full process once: mechanical design, electronics, 
fabrication, and software integration — and face every 
challenge that comes with building a real robotic system 
from scratch.

## Stack
- ROS2
- MoveIt
- Gazebo
- Onshape (URDF)
- 3D printing (FDM)
